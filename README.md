Bridge Foundry Web Site
=======================

This site is build with [Jeykll](https://jekyllrb.com/) and hosted on 
[Github Pages](https://pages.github.com/) -- pushing the master branch will
update the website.


## Local development

install Ruby

```
gem install bundler
bundle
```

to run the site:

```
jekyll serve
```

point your browser at: http://localhost:4000/
